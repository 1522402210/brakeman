Jenkins/Hudson Plugin

Jenkins (or Hudson) is a continuous integration tool.

A Brakeman plugin for Jenkins is available which will allow you to run Brakeman automatically when code changes and report the results.

Brakeman is a vulnerability scanner specifically designed for Ruby on Rails applications. It statically analyzes Rails application code to find security issues at any stage of development.
